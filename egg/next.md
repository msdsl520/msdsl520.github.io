# 5.洗发水死循环

老公给程序员老婆买了一瓶新的洗发水。于是，老婆在浴室待了一晚上。

因为洗发水的使用说明上写着：`揉出泡沫，冲水，然后重复`。

```c++
while(1){
    揉出泡沫;
    冲水;
}
```



于是，她在浴室待了一晚上，将整瓶洗发水都用完了。
